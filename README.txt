                javac *.java
How to run: java WebServer
            Then open broswer to http://localhost:8080/
HOw to stop: Pressing Crtl+ C

Features:
1. get request( could be text or images)
2. status code of 200, 304, 400, 403, 404
3. Last-Modified + If-MOdified-since
4. Connection status of "Keep-alive" & "close"
5. logging
